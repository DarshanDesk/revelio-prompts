# Content Improver

## Category
Writing

## Description
Enhances existing content for clarity, engagement, and professionalism.

## Prompt
```
Please improve the following content by:

1. Enhancing clarity and readability
2. Strengthening the message and impact
3. Correcting grammar and style issues
4. Making it more engaging for the target audience
5. Ensuring consistent tone throughout

Target audience: [DESCRIBE AUDIENCE]
Desired tone: [DESCRIBE TONE]

Original content:
[INSERT CONTENT HERE]

Please provide both the improved version and a brief explanation of key changes made.
```

## Usage Tips
- Specify the context and purpose of the content
- Mention any style preferences or constraints
- Include word count targets if applicable
- Note any specific sections that concern you

## Examples
Ideal for polishing emails, reports, presentations, marketing copy, or any written content.
