# Blog Post Outline Generator

## Category
Writing

## Description
Creates structured outlines for blog posts with engaging hooks and clear flow.

## Prompt
```
Create a detailed blog post outline for the following topic:

Topic: [YOUR TOPIC]
Target audience: [DESCRIBE AUDIENCE]
Desired tone: [e.g., professional, casual, educational]
Word count target: [e.g., 1000-1500 words]

Please include:
1. A compelling headline (with 2-3 alternatives)
2. An engaging introduction hook
3. Main sections with key points
4. Supporting examples or data needs
5. A strong conclusion with call-to-action
6. SEO keywords to incorporate

Format the outline with clear headings and bullet points.
```

## Usage Tips
- Be specific about your target audience for better tailored content
- Mention any key points or messages you want to include
- Specify the style guide if you have one
- Include competitor articles for reference if needed
