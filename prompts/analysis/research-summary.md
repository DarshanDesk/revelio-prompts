# Research Summary Generator

## Category
Analysis

## Description
Creates concise, structured summaries of research papers or technical documents.

## Prompt
```
Please create a comprehensive summary of the following research/document:

Title: [DOCUMENT TITLE]
Source: [SOURCE/LINK IF AVAILABLE]

Content:
[INSERT CONTENT OR KEY EXCERPTS]

Please structure the summary with:
1. Main objective/research question
2. Methodology overview
3. Key findings
4. Important conclusions
5. Practical implications
6. Limitations or caveats
7. Related areas for further exploration

Keep the summary concise but comprehensive, suitable for [TARGET AUDIENCE].
```

## Usage Tips
- Specify your field or context for appropriate terminology
- Indicate if you need it for a specific purpose (presentation, decision-making, etc.)
- Mention if you need focus on specific aspects
- Include any critical sections that must be covered

## Examples
Great for literature reviews, staying updated with research, preparing presentations, or making informed decisions.
