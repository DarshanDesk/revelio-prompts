# Data Insights Analyzer

## Category
Analysis

## Description
Extracts meaningful insights and patterns from data sets with actionable recommendations.

## Prompt
```
Analyze the following data and provide insights:

Data description: [DESCRIBE YOUR DATA]
Context: [PROVIDE BUSINESS/RESEARCH CONTEXT]
Key questions: [WHAT DO YOU WANT TO LEARN?]

Data:
[INSERT DATA OR SUMMARY]

Please provide:
1. Key patterns and trends identified
2. Notable outliers or anomalies
3. Correlations or relationships
4. Actionable insights
5. Recommendations based on findings
6. Suggestions for further analysis

Focus on practical, actionable insights.
```

## Usage Tips
- Provide context about what the data represents
- Specify what decisions or actions depend on this analysis
- Include data dictionary or variable descriptions if needed
- Mention any known limitations or biases in the data

## Examples
Useful for business analytics, research analysis, performance metrics review, or trend identification.
