# Meeting Notes Organizer

## Category
Productivity

## Description
Transforms raw meeting notes into structured, actionable summaries.

## Prompt
```
Please organize and structure my meeting notes:

Meeting topic: [MEETING TOPIC]
Date: [DATE]
Attendees: [LIST ATTENDEES]

Raw notes:
[PASTE YOUR NOTES HERE]

Please create a structured summary with:
1. Key discussion points
2. Decisions made
3. Action items (with owners and deadlines)
4. Open questions or issues
5. Follow-up needed
6. Important dates or milestones mentioned

Format it professionally and highlight critical action items.
```

## Usage Tips
- Include all notes even if messy - the AI will organize them
- Mention the meeting's purpose for better context
- Specify if certain information is confidential
- Indicate the intended audience for the notes

## Examples
Perfect for team meetings, client calls, standup notes, or any collaborative discussions requiring follow-up.
