# Task Prioritization Helper

## Category
Productivity

## Description
Helps prioritize tasks using proven frameworks for maximum productivity.

## Prompt
```
Help me prioritize the following tasks using the Eisenhower Matrix (Urgent-Important framework):

My tasks:
[LIST YOUR TASKS]

Context:
- Current goals: [YOUR MAIN GOALS]
- Deadlines: [ANY UPCOMING DEADLINES]
- Constraints: [TIME, RESOURCES, ETC.]

Please:
1. Categorize each task into:
   - Urgent & Important (Do First)
   - Important but Not Urgent (Schedule)
   - Urgent but Not Important (Delegate)
   - Neither Urgent nor Important (Eliminate)

2. Provide a prioritized action plan for today/this week
3. Suggest tasks that could be delegated or eliminated
4. Recommend time blocks for focused work
5. Identify potential bottlenecks or dependencies

Help me focus on what truly matters.
```

## Usage Tips
- Be honest about your capacity and energy levels
- Include context about your role and responsibilities
- Mention any recurring tasks or commitments
- Update regularly as priorities shift

## Examples
Great for daily planning, weekly reviews, project management, or overcoming overwhelm.
