# Story Prompt Generator

## Category
Creative

## Description
Creates engaging story prompts or develops story ideas into full narratives.

## Prompt
```
Create a compelling story based on:

Genre: [SPECIFY GENRE - sci-fi, mystery, romance, etc.]
Theme: [CENTRAL THEME OR MESSAGE]
Setting: [TIME AND PLACE]
Main character trait: [KEY CHARACTERISTIC]

Please provide either:
A) A detailed story prompt with:
   - Opening hook
   - Main conflict
   - Character dynamics
   - Potential plot twists
   - Ending possibilities

OR

B) A complete short story (500-1000 words) incorporating these elements

Make it engaging, original, and emotionally resonant.
```

## Usage Tips
- Specify the length and format you want
- Mention any themes or topics to avoid
- Include inspirations or similar stories you admire
- Define the target audience (age group, interests)

## Examples
Ideal for creative writing exercises, story development, narrative design, or overcoming writer's block.
