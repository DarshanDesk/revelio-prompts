# Code Review Assistant

## Category
Development

## Description
A prompt for conducting thorough code reviews with focus on best practices, security, and maintainability.

## Prompt
```
You are an experienced software engineer conducting a code review. Please review the following code and provide feedback on:

1. Code quality and readability
2. Potential bugs or edge cases
3. Security vulnerabilities
4. Performance considerations
5. Best practices and design patterns
6. Documentation and comments

Please be constructive and specific in your feedback, suggesting improvements where applicable.

Code to review:
[INSERT CODE HERE]
```

## Usage Tips
- Paste the code you want reviewed in place of [INSERT CODE HERE]
- Specify the programming language for context-specific feedback
- Mention any specific concerns you have about the code
- Include relevant context about the project's requirements

## Examples
Great for reviewing pull requests, learning from code feedback, or improving code quality before deployment.
