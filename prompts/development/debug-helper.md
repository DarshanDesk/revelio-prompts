# Debug Helper

## Category
Development

## Description
A systematic approach to debugging code issues with detailed analysis.

## Prompt
```
I'm encountering a bug in my code. Please help me debug it systematically:

1. Analyze the code logic and identify potential issues
2. Suggest debugging steps to isolate the problem
3. Recommend fixes with explanations
4. Provide prevention strategies for similar issues

Bug description: [DESCRIBE THE BUG]

Code:
[INSERT CODE HERE]

Error message (if any):
[INSERT ERROR MESSAGE]
```

## Usage Tips
- Provide clear description of expected vs actual behavior
- Include complete error messages and stack traces
- Share relevant context about the environment (OS, versions, etc.)
- Mention what you've already tried

## Examples
Useful for debugging runtime errors, logical bugs, performance issues, or unexpected behavior.
